# WhatWebFX

Aplikazioa Windows-erako eta Linux-erako (Debian GNU/LINUX eta deribatuak) eskuragarri dago.

## Aurrebaldintzak

### Windows

WSL instalatuta izan:
https://docs.microsoft.com/en-us/windows/wsl/

WhatWeb instalatuta izan:
https://github.com/urbanadventurer/WhatWeb

Wix Toolset instalatuta izan:
https://wixtoolset.org/releases/

### Linux

WhatWeb instalatuta izan:
https://github.com/urbanadventurer/WhatWeb

## Instalazioa

Defektuz .whatwebfx/ karpeta sortuko du, beharrezko fitxategi guztiekin zure erabiltzaile karpetan, beraz ez duzu ezer aldatu behar instalazioa burutzeko.

### Windows

1. Deskargatu Windows-erako instalatzailea
2. Bi click instalatzailean, eta Ados guztiari

### Linux

1. sudo dpkg -i linux_instalatzailea.deb
2. Ados guztiari
